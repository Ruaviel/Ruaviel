Ruaviel is a humanitarian stability system for people facing housing insecurity or financial hardship.
It replaces punitive housing and repayment structures with covenant-based, non-punitive stability mechanisms that preserve dignity during hardship events.
Through transparent and automated rules, Ruaviel ensures that individuals are never punished, reset, or destabilized due to inability to pay — creating a foundation of continuity, fairness, and long-term stability.



Phase 1 Pilot (MVP)


Goal


Build a minimal on-chain system that replaces “punitive housing logic” with contribution-based stability tracking and protected hardship states.



What We Are Building


A simple smart-contract system that:


1. Contribution Ledger (core feature)


Tracks resident contributions as stability credits instead of rent enforcement


2. Hardship State


Allows users to enter a protected state where:

•	no penalties
•	no resets
•	no loss of progress


3. Stability Continuity Rule


All prior contributions are preserved regardless of hardship events


Why This Matters


This pilot tests whether:

•	stability can be tracked without punitive enforcement
•	hardship protection can be embedded in code
•	housing-style systems can operate without coercion


User Flow (Simplified)


1.	User joins system
2.	Makes contributions when able
3.	System records “stability credits”
4.	User can enter hardship state anytime
5.	System pauses enforcement but preserves progress
6.	Credits accumulate toward long-term stability
